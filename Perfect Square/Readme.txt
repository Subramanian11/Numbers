A perfect square is a number, from a given number system, that can be expressed as the square of a number from the same number system.

Example:

Input: 4

Output: Yes

Explanation:  2*2=4
              when we are multiplying two same numbers,
              if it results the given input number, then it is perfect square
              other wise not.

Input: 8

Output: No

Explanation:  it is not possible to get like the previous.
              when we are multiplying two same numbers,
              if it results the given input number, then it is perfect square
              other wise not.

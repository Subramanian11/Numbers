A palindromic number is a number that remains the same when its digits are reversed.
eg:

input: 121

output: yes it's a palindrome number

explanation: The input number 121 is reveresed then we get the same number. so it is a palindrome number.

input: 420

output: no it's not a palindrome number

explanation: The input number 420 is reversed then we get 024, its not matching with the given input. so it is not a palindrome number.
